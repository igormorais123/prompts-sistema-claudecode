Arvore de Decisao

Preciso ler/modificar arquivo?
→ Ler: Read  | Editar: Edit  | Criar: Write
→ Nunca cat, sed, awk via Bash
Preciso encontrar algo no codigo?
→ Por nome: Glob  | Por conteudo: Grep
→ Busca aberta: Task/Explore
Preciso executar comando?
→ git, npm, docker: Bash
→ Longo: Bash (background)
Tarefa complexa ou incerta?
→ Plan Mode antes de implementar
2 falhas seguidas?
→ /clear + reformular prompt
